# fabich

The Programs you have to program at the end of Chapter 3 of the "The Rust Programming Language" Book

## Installation:
```sh
#Clone the git repo
sudo git clone https://github.com/revellan/fabich
cd fabich

#build from source
make

#install to /usr/local/bin
sudo make install
```

## Uninstall:
```sh
sudo rm /usr/local/bin/fabich
```
